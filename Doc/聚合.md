## 聚合

#### count
- 返回文档数量  

返回文档总数  
> db.c.count()

返回 usr 中 age = 8 的总数
>db.c.count({age: 8}) 


#### distinct
返回指定键所有的值形成的数组

```javascript
{"name" : "iphone", "type": 5}
{"name": "Mac", "type": 2013}

db.runCommand({"distinct":"people","key":"type"})

// => {"value":[5, 2013] ...}
 // or use
db.people.distinct('type')
// => [5, 2013]
